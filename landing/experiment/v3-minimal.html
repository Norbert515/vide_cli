Agents

Agent Role Description
orchestrator lead Delegates tasks, synthesizes results. Never writes code.
researcher context Explores codebase, finds patterns, gathers context.
implementer code Writes code in isolated git worktrees.
planner strategy Creates implementation plans before coding.
tester QA Runs Flutter apps, takes screenshots, validates with vision AI.
reviewer quality Reviews code for quality and security.

THE PROBLEM

One Agent Is a Bottleneck

⏳

Sequential Execution

Traditional AI assistants do one thing at a time. Research, then implement, then test. Each step waits for the last.

🔄

Context Lost

Long conversations lose context. The AI forgets what it learned 50 messages ago.

🚫

No Specialization

One agent tries to be everything: researcher, coder, tester, reviewer. Jack of all trades, master of none.

THE SOLUTION

A Team, Not a Tool

Vide spawns specialized agents that work in parallel. While the researcher explores your codebase, the implementer is already writing code in an isolated git worktree.

spawnAgent("researcher", "implementer", "tester")
orchestrator delegating
researcher exploring
implementer coding
tester validating
